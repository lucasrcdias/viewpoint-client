# viewpoint-client

## Comandos para rodar o projeto

### Instalar o nodejs
`curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -`

`sudo apt-get install -y nodejs`

`sudo apt-get install -y build-essential`

### Instalar as dependencias
` npm install -g bower `

` npm install -g gulp `

`npm install`

`bower install`

#### Rodar o gulp
` gulp `
